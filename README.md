# Remitly Landing Page

A modern, responsive landing page for rem built with HTML and CSS only.

## Features

- **Responsive Design**: Adapts to different screen sizes
- **Modern UI**: Clean and professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and interactive components
- **Full Navigation**: Complete navigation bar with dropdown menus
- **Hero Section**: Engaging hero section with call-to-action buttons
- **Feature Sections**: Detailed feature showcase with icons and descriptions
- **Testimonials**: Social proof section with customer testimonials
- **Footer**: Comprehensive footer with links and social media
